### Overall Directory Structure

At a high level, the structure looks roughly like this:

```
repo-dir
├── app
│   ├── assets
│   ├── languages
│   ├── src
│   ├── styles
│   ├── vendor
│   ├── 404.html
│   ├── config.js
│   └── index.html
├── build
│   ├── karma.conf.js
│   ├── protractor.conf.js
│   └── ...
├── bower.json
├── gulpfile.js
└── package.json
```
